## 개발 경험

- **Spring Boot** + **WebSquare** 기반 엔터프라이즈 개발 경험 보유 (주 사용 스택)
- Java 기반 백엔드, WebSquare(IBSheet 계열 UI 프레임워크) 기반 프론트엔드

## 학습 목표 (carmaint 프로젝트)

- **Next.js** (프론트엔드) / **Nest.js** (백엔드) — 개발 경험 전무, 이 프로젝트로 처음 학습
- TypeScript 생태계 전반 습득 목적

## 코드 작성 가이드라인

- 코드 내에 **학습 보조 주석** 적극 추가 요청
- Spring Boot 개념과 대응 관계를 주석에 명시하면 이해에 도움됨
  - 예: `@Module()` ≈ `@SpringBootApplication`, `@Injectable()` ≈ `@Service`, `@Controller()` ≈ `@RestController`
- WebSquare 대신 React/Next.js 컴포넌트 패턴으로 전환 중이므로 프론트엔드 개념 설명도 병행

**How to apply:** 코드 작성 시 WHY가 명백해도 처음 마주치는 패턴에는 한국어 주석으로 Spring Boot 유사 개념을 연결해 줄 것. 단, 자명한 반복 코드에는 생략.

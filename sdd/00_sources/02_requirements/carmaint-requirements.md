- SVG 일러스트 (부품 구조/역할 시각화)
- 부품 역할 설명 (한국어, HTML)
- 교환 통계 4칸: 교환 주기 / 다음 교환일 / 예상 km / 남은 날짜

## 5. 핵심 비즈니스 로직

### 다음 교환 계산
```
if pkm:
    next_km = last_km + pkm
    next_date = today + (next_km - current_km) / monthly_km * 30days

if pmo:
    next_date = last_date + pmo months
    next_km = current_km + months_diff(today, next_date) * monthly_km
```

### km 기반 예상 날짜 변환
```
months_until = (target_km - current_km) / monthly_km
target_date = today + months_until
```

## 6. 특수 케이스

- **타이밍체인**: pkm=0, isChain=true → 교환 일정 없음, 간트에 "교체 불필요" 배너 표시
- **NX4 전용 항목**: nx4=true → 특수 경고 배너 및 태그 표시
- **최근 교환 km 없이 날짜만 있는 경우**: pmo 방식으로 계산
- **출고 시점 기준 항목**: lkm=0 또는 ldt=차량 출고일 사용

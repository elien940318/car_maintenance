## SCR-01: 메인 대시보드

### 영역 구성 (위→아래)

```
┌─────────────────────────────────────────────┐
│ 헤더 (차량 정보)                              │
├─────────────────────────────────────────────┤
│ 차량 전용 배너 (NX4 HEV 등 특이사항)          │
├─────────────────────────────────────────────┤
│ 뷰 탭: [📊 간트 차트] [📋 목록 보기]          │
├─────────────────────────────────────────────┤
│ 알림 카드 (urgent/soon 항목)                  │
├─────────────────────────────────────────────┤
│ 힌트 바 ("클릭하면 상세 설명 볼 수 있습니다")   │
├─────────────────────────────────────────────┤
│ 간트 차트 뷰 또는 목록 테이블 뷰              │
├─────────────────────────────────────────────┤
│ 정보 카드 그리드 (부품 지식 카드)             │
└─────────────────────────────────────────────┘
```

### 헤더 컴포넌트

| 요소 | 내용 |
|------|------|
| 차량 태그 | `차량명 · 연식` (monospace, 태그 스타일) |
| 제목 | `내 차 교환 일정` |
| 메타 배지 | 현재 km / 월 평균 km / 기준일 / 최근 교환 요약 |

### 간트 차트 컴포넌트

| 요소 | 설명 |
|------|------|
| X축 헤더 | 3년(36개월) / 3개월 단위 눈금 / 연도 변경 시 강조 |
| TODAY 수직선 | 민트색 실선 + "TODAY" 레이블 |
| 섹션 레이블 | 카테고리명 (점선 구분선) |
| 행(Row) | 좌측: 부품명·서브태그·D-day / 우측: 바 |
| 바 색상 | 카테고리별 고유 색상 (engine=mint, hybrid=cyan, 등) |
| 바 상태 | done(불투명), next(반투명+테두리), future(저불투명) |
| 체인 배너 | 패턴 배경 "교체 불필요" 배너 (isChain 항목) |
| 날짜 핀 | 다음 교환 예정월 레이블 |

### 목록 테이블 컴포넌트

| 컬럼 | 내용 |
|------|------|
| 항목 | 부품명 (색상) + 서브태그 (회색) + 배지 (NX4·CHAIN) |
| 주기 | `7,500km` 또는 `6개월` 형식 |
| 최근 교환 | 날짜 또는 km |
| 다음 교환 예정일 | 날짜 (상태 색상) |
| 예상 km | km 형식 |
| 상태 | 알약 배지: URGENT(rose) / SOON(amber) / 여유(green) / CHAIN(cyan) |

### 알림 카드 컴포넌트

- 카드 상단 3px 컬러 바: urgent=rose, soon=amber
- 배지 레이블: 임박 / 주의
- 부품명 (카테고리 색상)
- 예정일 (monospace)
- 남은 날짜 또는 초과 일수

## SCR-02: 부품 상세 사이드 패널

### 레이아웃

```
┌────────────────────────────────┐
│ [태그]  부품명          [✕ 닫기] │
│ 서브 태그 · 다음 교환 D-day      │
│                                 │
│ ┌──────────────────────────┐   │
│ │   SVG 일러스트 (부품 구조)  │   │
│ └──────────────────────────┘   │
│                                 │
│ 🔩 이 부품의 역할                │
│ [역할 설명 텍스트]               │
│                                 │
│ 📅 내 차 교환 정보              │
│ ┌──────────┬──────────┐        │
│ │ 교환 주기 │ 다음 교환  │        │
│ ├──────────┼──────────┤        │
│ │ 예상 km  │ 남은 날짜  │        │
│ └──────────┴──────────┘        │
│                                 │
│ 💡 정비 팁 (배경 민트)           │
└────────────────────────────────┘
```

### 패널 동작

- 오른쪽 슬라이드-인 (transition 0.28s cubic-bezier)
- 배경 오버레이 (blur 2px)
- 오버레이 클릭 or ✕ 버튼으로 닫기

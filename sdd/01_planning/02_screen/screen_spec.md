# 화면 명세

> 01_planning/02_screen: 화면 구성과 컴포넌트 설명. 반응형 분기 기준: 640px.

---

## 화면 목록

| ID | 화면명 | 설명 |
|----|--------|------|
| SCR-01 | 메인 대시보드 | 차량 헤더 + 뷰 (모바일: 티켓 카드 / 태블릿+: 간트·테이블) |
| SCR-02 | 부품 상세 패널 | 모바일: 바텀 시트 / 태블릿+: 우측 사이드 패널 (교환완료 입력 포함) |
| SCR-03 | 차량 등록·수정 | 차량 제원 입력 + 프리셋 선택 폼 |
| SCR-04 | 정비 기록 일괄 입력 | 다수 부품 일괄 교환 기록 (Phase 2) |

## SCR-01: 메인 대시보드

### 빈 상태 (차량 미등록)

등록된 차량이 없을 때 대시보드 대신 아래 화면을 표시한다.

```
┌──────────────────────────────────────┐
│  carmaint                            │
├──────────────────────────────────────┤
│                                      │
│          🚗                          │
│   차량 정보를 먼저 등록해주세요.       │
│   제원을 입력하면 교환 일정이          │
│   자동으로 계산됩니다.                │
│                                      │
│   [ ＋ 차량 등록하기 ]               │
│                                      │
└──────────────────────────────────────┘
```

- "차량 등록하기" 버튼 탭 → SCR-03으로 이동
- SCR-03 등록 완료 후 → SCR-01(대시보드)로 자동 이동
- 헤더·배너 없이 단순 안내 화면만 표시

---

### 반응형 레이아웃 분기

| 뷰포트 | 뷰 구성 |
|--------|--------|
| 모바일 (< 640px) | 헤더 → 배너 → **티켓 카드 목록** (알림 카드·탭 없음) |
| 태블릿+ (≥ 640px) | 헤더 → 배너 → 탭(간트/테이블) → 알림 카드 → 간트 또는 테이블 → 정보 카드 그리드 |

---

### 모바일 레이아웃 (< 640px)

```
┌──────────────────────────────────────┐
│  헤더 (차량명 태그 · 현재km · 기준일)  │  ← 최근교환 배지 없음
├──────────────────────────────────────┤
│  차량 전용 배너 (isVehicleSpecific)   │  ← 해당 차량만 표시
├──────────────────────────────────────┤
│  카테고리 섹션 레이블                 │
│  ┌────────────────────────────────┐  │
│  │  티켓 카드 (urgent → rose)      │  │
│  └────────────────────────────────┘  │
│  ┌────────────────────────────────┐  │
│  │  티켓 카드 (soon → amber)       │  │
│  └────────────────────────────────┘  │
│  ┌────────────────────────────────┐  │
│  │  티켓 카드 (ok → green, 연한)   │  │
│  └────────────────────────────────┘  │
│  … (카테고리별 반복)                  │
└──────────────────────────────────────┘
```

- 알림 카드 섹션 없음 (티켓 카드 색상이 긴급도를 표현)
- 간트 차트 없음 (좁은 화면에 비효율적)
- 탭 없음 (티켓 카드 단일 뷰)

### 태블릿+ 레이아웃 (≥ 640px)

```
┌─────────────────────────────────────────────┐
│  헤더 (차량명 태그 · 현재km · 기준일)          │  ← 최근교환 배지 없음
├─────────────────────────────────────────────┤
│  차량 전용 배너                               │
├─────────────────────────────────────────────┤
│  [📊 간트 차트]  [📋 목록 보기]  ← 뷰 탭     │
├─────────────────────────────────────────────┤
│  알림 카드 (urgent / soon)                   │
├─────────────────────────────────────────────┤
│  힌트 바 ("항목 클릭 시 상세 확인 가능")       │
├─────────────────────────────────────────────┤
│  간트 차트 뷰  또는  목록 테이블 뷰           │
├─────────────────────────────────────────────┤
│  정보 카드 그리드 (부품 지식)                 │
└─────────────────────────────────────────────┘
```

---

### 헤더 컴포넌트

| 요소 | 내용 |
|------|------|
| 차량 태그 | `차량명 · 연식 · 연료` (monospace, 태그 스타일) |
| 제목 | `내 차 교환 일정` |
| 메타 배지 | 현재 km / 기준일 |

> **삭제**: 최근 교환 요약 배지 (교환 정보는 각 부품 카드에서 확인 가능)

---

### 티켓 카드 컴포넌트 (모바일 전용)

상태에 따라 카드 좌측 4px 컬러 바와 배경 tint로 긴급도를 표현한다.

```
┌──────────────────────────────────────────────┐
▌  엔진오일        [NX4]        D-45  🔴 임박  ▌  ← 좌측 4px rose 바
│  7,500km마다  →  2026-07-29 예정              │
│  최근: 2026-06-07 · 89,485km                 │
└──────────────────────────────────────────────┘

┌──────────────────────────────────────────────┐
▌  에어컨필터                   D-120  🟡 주의  ▌  ← 좌측 4px amber 바
│  6개월마다  →  2026-12-07 예정               │
│  최근: 2026-06-07                            │
└──────────────────────────────────────────────┘

┌──────────────────────────────────────────────┐
▌  에어클리너                   D-430  🟢 여유  ▌  ← 좌측 4px green 바
│  40,000km마다  →  2027-09-15 예정            │
│  최근: 2026-06-07 · 89,485km                │
└──────────────────────────────────────────────┘

┌──────────────────────────────────────────────┐
▌  타이밍체인      [CHAIN]       교체 불필요    ▌  ← 좌측 4px cyan 바
│  모니터링만 필요                              │
└──────────────────────────────────────────────┘
```

#### 상태별 색상 규칙

| 상태 | 좌측 바 | 배경 tint | D-day 텍스트 | 아이콘 |
|------|---------|----------|------------|-------|
| urgent (< 90일 또는 초과) | `--rose` #f87171 | rose/8% | rose | 🔴 |
| soon (90~179일) | `--amber` #fbbf24 | amber/8% | amber | 🟡 |
| ok (≥ 180일) | `--green` #22c55e | green/5% | muted | 🟢 |
| chain (교환 불필요) | `--cyan` #38bdf8 | cyan/5% | cyan | — |

#### 카드 요소

| 영역 | 내용 |
|------|------|
| 1행 좌 | 부품명 (bold) + 차량 전용 태그 `[NX4]` (amber) 또는 `[CHAIN]` (cyan) |
| 1행 우 | D-day 숫자 + 상태 아이콘·텍스트 |
| 2행 | `교환 주기` → `다음 교환 예정일 · 예상 km` (카테고리 색상) |
| 3행 | 최근 교환 날짜 · km (muted, chain은 생략) |
| 탭 동작 | 바텀 시트 오픈 (SCR-02) |

#### 카드 섹션 구분

카테고리별로 섹션 레이블(점선 구분선 + 카테고리명)로 그룹화.

---

### 간트 차트 컴포넌트 (태블릿+ 전용)

| 요소 | 설명 |
|------|------|
| X축 헤더 | 3년(36개월) / 3개월 단위 눈금 / 연도 변경 시 강조 |
| TODAY 수직선 | 민트색 실선 + "TODAY" 레이블. 초기 진입 시 자동 스크롤 앵커 |
| 섹션 레이블 | 카테고리명 (점선 구분선) |
| 행(Row) | 좌측: 부품명·서브태그·D-day / 우측: 바 |
| 바 색상 | 카테고리별 고유 색상 (engine=mint, hybrid=cyan 등) |
| 바 상태 | done(불투명), next(반투명+테두리), future(저불투명) |
| 체인 배너 | 패턴 배경 "교체 불필요" 배너 |
| 날짜 핀 | 다음 교환 예정월 레이블 |

---

### 목록 테이블 컴포넌트 (태블릿+ 전용)

| 컬럼 | 내용 |
|------|------|
| 항목 | 부품명 (색상) + 서브태그 (회색) + 배지 |
| 주기 | `7,500km` 또는 `6개월` 형식 |
| 최근 교환 | 날짜 또는 km |
| 다음 교환 예정일 | 날짜 (상태 색상) |
| 예상 km | km 형식 |
| 상태 | 알약 배지: URGENT(rose) / SOON(amber) / 여유(green) / CHAIN(cyan) |

---

### 알림 카드 컴포넌트 (태블릿+ 전용)

- 카드 상단 3px 컬러 바: urgent=rose, soon=amber
- 배지 레이블: 임박 / 주의
- 부품명 (카테고리 색상)
- 예정일 (monospace) + 남은 날짜 또는 초과 일수

## SCR-02: 부품 상세 패널 (반응형)

### 태블릿+ (≥ 640px) — 우측 사이드 패널

```
┌─────────────────────────────────────┐
│ [카테고리 태그]  부품명      [✕ 닫기] │
│ 서브 태그 · 다음 교환 D-day           │
│                                      │
│ ┌───────────────────────────────┐   │
│ │   SVG 일러스트 (부품 구조)      │   │
│ └───────────────────────────────┘   │
│                                      │
│ 🔩 이 부품의 역할                     │
│ [역할 설명 텍스트]                    │
│                                      │
│ 📅 교환 정보              │
│ ┌────────────┬────────────┐         │
│ │  교환 주기  │  다음 교환  │         │
│ ├────────────┼────────────┤         │
│ │  예상 km   │  남은 날짜  │         │
│ └────────────┴────────────┘         │
│                                      │
│ 💡 정비 팁                           │
│                                      │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│ ✅ 교환완료 기록                      │  ← 하단 고정 영역
│  교환 날짜   [ 2026-06-14      ▾ ]  │
│  교환 km    [ 89,660            ]   │
│  메모       [ 예: 현대 서비스센터 ]   │
│ ┌────────────────────────────────┐  │
│ │          저장하기               │  │
│ └────────────────────────────────┘  │
└─────────────────────────────────────┘
```

- 오른쪽 슬라이드-인 (transition 0.28s cubic-bezier)
- 배경 오버레이 (blur 2px), 오버레이 클릭 or ✕ 버튼으로 닫기
- 교환완료 입력 영역은 패널 하단에 항상 고정 노출 (별도 모달 없음)
- 기본값: 교환 날짜 = 오늘 / 교환 km = 차량 현재 km

### 모바일 (< 640px) — 바텀 시트

```
┌───────────────────────────────────────┐
│            (배경 오버레이)               │
├───────────────────────────────────────┤
│              ━━━ (핸들)                │  ← 스와이프 다운으로 닫기
│ [태그]  부품명                  [✕]    │
│ 서브 태그 · D-day                      │
│ ─────────────────────────────────     │  ↑ 내부 스크롤 영역
│  SVG 일러스트                          │
│ 🔩 부품 역할 설명                       │
│ 📅 교환 정보 (2×2 그리드)              │
│ 💡 정비 팁                            │
│ ─────────────────────────────────     │  ↓ 하단 고정 영역
│ ✅ 교환완료 기록                        │
│  날짜   [ 2026-06-14 ▾ ]              │
│  km     [ 89,660      ]               │
│  메모   [               ]             │
│ [          저장하기           ]        │
└───────────────────────────────────────┘
```

- 하단 슬라이드업 (max-height: 75vh, 내부 스크롤)
- 교환완료 입력 영역은 바텀 시트 하단 고정 (별도 모달 없음)
- 기본값: 교환 날짜 = 오늘 / 교환 km = 차량 현재 km

## SCR-03: 차량 등록·수정

### 단계 구성

```
Step 1 — 기본 정보
  차량명 (자유 입력)
  연식

Step 2 — 제원 선택
  차종   [ 중형 SUV ▾ ]
  연료   [ 하이브리드(HEV) ▾ ]
  변속기 [ 자동변속기(AT) ▾ ]

Step 3 — 주행 정보
  현재 주행거리  [ 89,660 ] km
  연간 주행거리  [ 9,000  ] km/년   ← 월 평균은 내부에서 ÷12 자동 계산
  기준일         [ 2026-06-14 ]

Step 4 — 정비 항목 선택 (프리셋 제안)
  ☑ 엔진오일       10,000 km  (수정 가능)
  ☑ 오일필터       10,000 km
  ☑ 스파크플러그   60,000 km
  ☐ 타이밍벨트     (제외됨 — HEV는 체인)
  ...
  [   완료 — 차량 등록   ]
```
